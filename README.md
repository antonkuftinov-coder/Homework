Шаг 1 - откройте папку проекта
1.1 Скачайте с сайта проекта server.py
1.2 Комбинацией клавиш «Win+R» открываем выполнитель и вводим команду «cmd». После нажимаем ОК или клавишу на клавиатуре «ENTER».
1.3	Проверяем версию питона командой «python --version» для Windows и «python3 --version» для Linux (желательно иметь версию третью и больше).

Шаг 2 - Открыть терминал
Переходим в нашу папку используя команду CD

Шаг 3 - Установить Flask
Далее нам необходимо установить библиотеку «flask» для работы с веб-приложениями — «pip install flask»

Шаг 4 - Запустить сервер
Pапускаем cервер командой «python server.py».

Шаг 5 - Открыть сайт в непосредственном браузере
Не закрывая консоль переходим по адрессу:
http://localhost:5000

Установка состояния
Состояние можно меня через нажатие кнопки
непосредственно на самом сайте или используя данные сылки:
http://localhost:5000/on
http://localhost:5000/off
