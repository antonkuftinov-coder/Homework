Шаг 1 - откройте папку проекта
1.1 Скачайте с сайта проекта server.py
1.2 Откройте VS Code
1.3 Открыть в редакторе папку где лежит server.py

Шаг 2 - Открыть терминал
Открыть в VS Code терминадл комбинацией
клавиш CTR + `

Шаг 3 - Установить Flask
Ввести в терминал следущию команду:
pip install flask

Шаг 4 - Запустить сервер
ввести в терминал VS Code:
python server.py

Шаг 5 - Открыть сайт в непосредственном браузере
Переходим по адрессу:
http://localhost:5000
